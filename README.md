# Photography Portfolio

Professional photography portfolio built with Next.js and Strapi CMS. Initial UI scaffolded with v0.

## Stack

-   Next.js 15
-   TypeScript
-   Tailwind CSS
-   Strapi CMS
-   Vercel deployment

## Development

```bash
pnpm install
pnpm dev
```

## Roadmap
